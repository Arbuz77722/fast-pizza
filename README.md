## Project Overview

A simple pizza ordering application that allows users to browse a menu, add pizzas to a cart, and place an order. This project focuses on state management using Redux Toolkit and clean feature-based architecture.

---

## Features

- View pizza menu
- Add pizzas to cart
- Update item quantity
- Remove items from cart
- Create order
- Search existing order
- Automatic cart overview
- Geolocation support for delivery

---

## Tech Stack

- React
- Redux Toolkit (state management)
- Tailwind CSS
